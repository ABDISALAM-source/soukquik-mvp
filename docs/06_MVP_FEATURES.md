# Fonctionnalités du MVP

## Client
- [x] Inscription / connexion
- [x] Recherche produits + services (texte, catégorie, type, prix)
- [x] Voir la liste des boutiques et prestataires
- [x] Page détail boutique / service
- [x] Chat basique (boutique/prestataire)
- [x] Commander un produit
- [x] Réserver un service

## Vendeur
- [x] Créer/éditer/supprimer une boutique
- [x] Ajouter / modifier / supprimer un produit
- [x] Voir les commandes reçues et changer leur statut
- [x] Dashboard simple (nombre de commandes, revenu du jour, produits actifs)

## Prestataire de service
- [x] Créer/éditer son profil de service
- [x] Définir prix et zone d'intervention
- [x] Recevoir et gérer des réservations
- [x] Dashboard simple (réservations en attente, revenu du jour)

## Admin
- [x] Voir la liste des utilisateurs
- [x] Suspendre / réactiver un utilisateur ou une boutique/service (modération)
- [x] Statistiques simples (nb users, nb shops, nb services, nb commandes)

## Hors scope MVP (documenté pour V2)
- Paiement en ligne / mobile money
- Tracking GPS livreur en temps réel
- Recherche par image (IA)
- Animations "vagues" dynamiques basées sur le trafic live
- Notifications push
- Publicité sponsorisée payante
